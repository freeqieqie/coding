📘 养宠规则

🍖 饱腹值自然变化

  1. 饱腹值 100 → 70 时，每小时减少 1.25。
  2. 饱腹值 70 → 55 时，每小时减少 0.625。
  3. 饱腹值 55 → 19 时，每小时减少 0.5。
  4. 饱腹值 19 → 0 时，每小时减少 1。

⭐ 成长值自然变化

  1. 饱腹值 100 → 70 时，成长值每小时增加 1。
  2. 饱腹值 70 → 55 时，成长值不增加也不减少。
  3. 饱腹值 55 → 19 时，成长值每小时减少 1。
  4. 饱腹值 19 → 0 时，成长值每小时减少 2。
当宠物的饱腹值降到 0 时会死亡。成长值只影响等级变化，不决定死亡；最低只会降到 0 级 0 成长值。
